## 2. Annual Goal Management

### 2.1 Goal Structure

```typescript
interface AnnualGoal {
  id: string;
  function_id: string;
  year: number;

  title: string; // e.g., "Reduce operational costs"
  description: string;

  // Measurement
  target_metric: string; // "Cost reduction"
  target_value: string; // "-15%"
  current_value: string; // "-8.2%"
  measurement_method: string; // "YoY comparison of OpEx"
  unit: string; // "%"

  // Scoring
  measurement_type: 'formula' | 'discretionary' | 'hybrid';
  formula_weight: number; // 0-100
  discretionary_weight: number; // 0-100
  discretionary_score: number; // 0-100 (manual input)

  // Compensation
  bonus_weight: number; // % of total bonus (must total 100% across all goals)

  // Status
  status: 'draft' | 'active' | 'completed' | 'cancelled';
  completion_percentage: number; // Auto-calculated from quarterly goals
}
```

### 2.2 Measurement Types

#### **Formula-Based (Automated)**
- System calculates completion based on current_value vs target_value
- Examples: Revenue growth, cost reduction, headcount
- Formula: `completion = (current - baseline) / (target - baseline) * 100`

#### **Discretionary (Manual Scoring)**
- Admin manually assigns score 0-100 at review time
- Used for qualitative goals: "Improve team morale", "Vendor relationships"
- Requires review notes explaining score

#### **Hybrid (Mix of Both)**
- Combines formula and discretionary
- Example: 70% formula (cost metrics) + 30% discretionary (quality assessment)
- Final score: `(formula_score * formula_weight + discretionary_score * discretionary_weight) / 100`

## 7. Performance Scoring & Bonus Calculation

### 7.1 Scoring Logic

#### **Formula-Based Goals**
```python
def calculate_formula_score(current_value, baseline, target, unit):
    if unit == '%':
        # For percentage targets (e.g., 92% â†’ 98%)
        progress = (current_value - baseline) / (target - baseline)
        score = min(100, max(0, progress * 100))
    elif unit == '$':
        # For dollar targets (e.g., $2.5M â†’ $2.125M for cost reduction)
        progress = abs(current_value - baseline) / abs(target - baseline)
        score = min(100, max(0, progress * 100))
    elif unit == 'count':
        # For count targets (e.g., 5 vendors â†’ 3 vendors)
        progress = abs(baseline - current_value) / abs(baseline - target)
        score = min(100, max(0, progress * 100))

    return score
```

#### **Discretionary Goals**
```python
def calculate_discretionary_score(goal_id):
    # Admin manually enters score 0-100 during review
    score = get_discretionary_score_from_db(goal_id)

```

#### **Hybrid Goals**
```python
def calculate_hybrid_score(goal):
    formula_score = calculate_formula_score(...)
    discretionary_score = goal.discretionary_score

    total_score = (
        formula_score * goal.formula_weight / 100 +
        discretionary_score * goal.discretionary_weight / 100
    )

    return total_score
```

### 7.2 Overall Bonus Calculation

```python
def calculate_function_bonus(function_id, year, quarter=None):
    goals = get_annual_goals(function_id, year)

    total_weighted_score = 0
    total_weight = 0

    for goal in goals:
        # Get goal score based on measurement type
        if goal.measurement_type == 'formula':
            goal_score = calculate_formula_score(goal)
        elif goal.measurement_type == 'discretionary':
            goal_score = goal.discretionary_score
        else:  # hybrid
            goal_score = calculate_hybrid_score(goal)

        # Weight the score
        weighted_score = goal_score * goal.bonus_weight
        total_weighted_score += weighted_score
        total_weight += goal.bonus_weight

    # Final bonus percentage
    overall_score = total_weighted_score / total_weight
    bonus_multiplier = overall_score / 100  # 0.0 to 1.0

    # Can be over 100% (bonus multiplier > 1.0) if goals exceeded
    return {
        'overall_score': overall_score,
        'bonus_multiplier': bonus_multiplier,
        'goal_scores': [goal details]
    }
```

For bonus participants who are not function leads:

```python
def calculate_individual_bonus(user_id, function_id, year):
```
